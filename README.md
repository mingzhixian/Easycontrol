## 注意
 Gitee和GitHub代码将保持同步，请自行选择。

- [Github地址](https://github.com/mingzhixian/Easycontrol)

## 简介
本软件基于开源项目Scrcpy，对其进行了大量魔改，实现了其安卓客户端，并添加了一些功能，实现了安卓端控制安卓端。

## 功能特色
- 使用简单
- 支持音频传输
- 多设备连接
- 支持有线连接
- 多设备剪切板同步
- 多设备共享主控端物理键盘(需配合微信输入法或QQ输入法等输入中文)
- 启动迅速
- 低延迟
- 支持分辨率自适应
- 良好的旋转支持
- 支持小窗显示与全屏显示
